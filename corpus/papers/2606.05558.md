###### Abstract

Evaluating large language model (LLM) agents in multi-turn interactive environments is expensive and risky, as it requires online environment interaction. We propose Adwm (Autoregressive Diffusion World Model), an evaluation framework that estimates the performance of a new LLM agent policy purely from pre-collected trajectories. The core idea is to learn a latent diffusion world model that simulates how the environment responds to the evaluation policy, without ever executing it in the real environment. Existing diffusion-based OPE methods guide full trajectories in a single pass by jointly diffusing states and actions, an assumption that breaks down for LLM agents whose actions are discrete text that must be sampled from the policy after observing the environment. Unlike autoregressive world models that suffer from compounding errors, Adwm models each transition as an independent denoising process, enabling reliable step-by-step rollouts where the world model and agent alternate in causal order. Crucially, the LLM agent under evaluation directly guides the diffusion generation at each step via a policy-conditioned score function, ensuring that simulated trajectories accurately reflect its decision-making patterns. Empirically, Adwm achieves accurate value estimates and evaluation reliability across diverse multi-turn agent tasks, demonstrating its promise as a practical framework for offline LLM agent evaluation.

## 1 Introduction

Large language model agents are increasingly deployed in multi-turn interactive environments, navigating websites, writing and executing code, and reasoning over long document contexts (Yao et al., [2022](#bib.bib5 "React: synergizing reasoning and acting in language models"); Zhou et al., [2023](#bib.bib6 "Webarena: a realistic web environment for building autonomous agents"); Jimenez et al., [2023](#bib.bib7 "Swe-bench: can language models resolve real-world github issues?")). As these agents are trusted with higher-stakes tasks, evaluating a new agent policy before deployment becomes critical. Yet evaluation is expensive; every new agent must be executed live in the real environment, consuming API budget and potentially causing irreversible side effects.

Off-policy evaluation (OPE) addresses this challenge by estimating the value of a new agent from pre-collected offline data, without any further environment interaction. Importance sampling methods reweight offline trajectories by policy likelihood ratios (Liu et al., [2018](#bib.bib27 "Breaking the curse of horizon: infinite-horizon off-policy estimation"); Metelli et al., [2021](#bib.bib26 "Subgaussian and differentiable importance sampling for off-policy evaluation and learning")), but these weights grow exponentially with trajectory length, making the estimator impractical for multi-turn agentic settings. Value-based direct methods attempt to fit a value function on offline data (Le et al., [2019](#bib.bib22 "Batch policy learning under constraints"); Paine et al., [2020](#bib.bib34 "Hyperparameter selection for offline reinforcement learning")), but suffer from compounding bias when the evaluation policy differs substantially from the behavior policy. Doubly robust methods combine the two (Farajtabar et al., [2018](#bib.bib35 "More robust doubly robust off-policy evaluation"); Kallus and Uehara, [2022](#bib.bib28 "Efficiently breaking the curse of horizon in off-policy evaluation with double reinforcement learning")), partially mitigating these issues but not resolving the fundamental difficulty of long-horizon distribution shift in high-dimensional text spaces.

Among direct methods, a more promising model-based alternative is to build a world model: learn the environment dynamics from offline data and simulate rollouts under the evaluation policy (Hanna et al., [2017](#bib.bib36 "Bootstrapping with models: confidence intervals for off-policy evaluation"); Lu et al., [2023](#bib.bib30 "Synthetic experience replay")). In the agentic setting, this simulation must be _autoregressive_: each LLM action depends on the observation just received, so the world model and agent must alternate step by step in causal order. Autoregressive transformer-based world models (Micheli et al., [2022](#bib.bib13 "Transformers are sample-efficient world models"); Hafner et al., [2023](#bib.bib12 "Mastering diverse domains through world models")) follow this structure naturally, but generate observations token by token, causing errors to accumulate within each step and compound further across steps, a critical failure mode for the long-horizon tasks that LLM agents face. Diffusion models offer a principled remedy: by modeling each transition as an independent denoising process, errors do not propagate across steps (Ho et al., [2020](#bib.bib1 "Denoising diffusion probabilistic models"); Dhariwal and Nichol, [2021](#bib.bib2 "Diffusion models beat gans on image synthesis")), and score-function guidance enables the evaluation policy to steer rollouts without retraining. However, existing diffusion-based OPE methods (Jackson et al., [2024](#bib.bib4 "Policy-guided diffusion"); Lu et al., [2023](#bib.bib30 "Synthetic experience replay")) were designed for continuous control, where states and actions are both real-valued vectors that can be jointly noised and denoised as a single tensor in one pass, with actions generated by the diffusion process itself and the evaluation policy entering only as a guidance signal. In LLM agent settings, this assumption fundamentally breaks down: actions are discrete text that must be sampled from the LLM _after_ observing the environment and cannot participate in a joint continuous diffusion over trajectories. Generating full trajectories in a single pass therefore requires all LLM actions to be known before any observation is generated, a circular dependency that is irreconcilable with the step-by-step rollout that agentic evaluation demands.

(Middle) Traditional off-policy evaluation learns a model-based simulator from offline data, but suffers from two fundamental issues: distribution shift between the behavior and target policies, and compounding error accumulated over multi-step rollouts. (Right) Our AdwmAdwm addresses both issues simultaneously. We propose Adwm (Autoregressive Diffusion World Model), an offline evaluation framework for LLM agents that resolves this tension by instantiating the world model itself as a diffusion process (see Figure [1](#S1.F1 "Figure 1 ‣ 1 Introduction ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")). The key insight is that the globally policy-guided trajectory law can be _exactly_ factored into a sequence of single-step conditionals, each decomposing as a product of a prior, an action-posterior, and a policy continuation factor (Theorem [1](#Thmtheorem1 "Theorem 1 (Fused single-step conditional). ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")). This product structure maps directly onto guided diffusion, with the evaluation policy steering every denoising step via its log-likelihood gradient. Because each transition is an independent denoising process, errors do not compound across steps. Adwm is trained entirely offline and evaluates any new agent at inference without retraining.

Our contributions are summarized as follows:  
⊳ We derive the exact autoregressive single-step conditional induced by a policy-guided trajectory law, showing it decomposes into a principled three-factor product (Theorem [1](#Thmtheorem1 "Theorem 1 (Fused single-step conditional). ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")).  
⊳ We realize this decomposition as a guided diffusion process in a structured latent space, where the evaluation policy participates in every denoising step without retraining the world model.  
⊳ We demonstrate empirically that Adwm correctly ranks evaluation policies across diverse multi-turn LLM agent benchmarks, outperforming classic OPE baselines under realistic LLM importance ratios.

## 2 Related Work

#### Model-Based OPE.

Model-based OPE methods estimate policy value by learning a transition model from offline data and simulating rollouts under the evaluation policy (Yu et al., [2020](#bib.bib14 "Mopo: model-based offline policy optimization"), [2021](#bib.bib15 "Combo: conservative offline model-based policy optimization")). The core difficulty is compounding model error: small inaccuracies in the learned transition accumulate across steps, causing simulated trajectories to drift away from the true environment dynamics over long horizons (Janner et al., [2019](#bib.bib38 "When to trust your model: model-based policy optimization"); Sutton, [1996](#bib.bib39 "Model-based reinforcement learning with an approximate, learned model"); Asadi et al., [2019](#bib.bib42 "Combating the compounding-error problem with a multi-step model")). Existing approaches mitigate this through pessimistic value penalties or conservative policy optimization (Jin et al., [2021](#bib.bib43 "Is pessimism provably efficient for offline rl?")), but these techniques were designed for low-dimensional continuous control and do not extend to free-form text observations. Beyond error accumulation, these methods provide no mechanism for conditioning the simulated rollout on the evaluation policy, so generated trajectories reflect the behavior distribution rather than what the evaluation policy would actually produce (Voloshin et al., [2019](#bib.bib33 "Empirical study of off-policy policy evaluation for reinforcement learning"); Feng et al., [2020](#bib.bib32 "Accountable off-policy evaluation with kernel bellman statistics")). Adwm addresses both issues: modeling each transition as an independent denoising process breaks the error accumulation chain (Janner et al., [2022](#bib.bib23 "Planning with diffusion for flexible behavior synthesis"); Ajay et al., [2022](#bib.bib24 "Is conditional generative modeling all you need for decision-making?")), while the guided score function ensures every generated observation is conditioned on the evaluation policy.

#### Diffusion Models for World Modeling.

Diffusion models have emerged as powerful environment simulators, modeling each transition as an independent denoising process and thereby avoiding the compounding errors that plague autoregressive world models. Full-sequence models jointly generate entire trajectories (Janner et al., [2022](#bib.bib23 "Planning with diffusion for flexible behavior synthesis"); Ajay et al., [2022](#bib.bib24 "Is conditional generative modeling all you need for decision-making?"); Huang et al., [2025](#bib.bib18 "Vid2world: crafting video diffusion models to interactive world models")), treating states and actions as a single jointly diffused tensor. This design precludes step-by-step rollout, as actions are produced by the diffusion process itself rather than sampled from an external policy. Score-based guidance (Ho and Salimans, [2022](#bib.bib3 "Classifier-free diffusion guidance")) can steer such generation toward a target policy (Lu et al., [2023](#bib.bib30 "Synthetic experience replay"); Jackson et al., [2024](#bib.bib4 "Policy-guided diffusion")), but inherits the same limitation: actions must be real-valued vectors that participate in the joint diffusion, an assumption that breaks down for LLM agents whose actions are discrete text sampled from the policy after observing the environment. Adwm resolves this by deriving an exact factored score that enables the evaluation policy to steer each denoising step autoregressively, without requiring actions to be part of the diffusion process.

## 3 Problem Formulation

### 3.1 Agentic Reinforcement Learning

We formulate agentic reinforcement learning as a sequential decision-making problem, modeled as a Partially Observable Markov Decision Process (POMDP) ℳ\=(𝒮,𝒜,𝒪,P,ℰ,r,μ,γ), where 𝒮 is the state space, 𝒜 is the action space, 𝒪 is the observation space, P:𝒮×𝒜→Δ​(𝒮) is the transition kernel, ℰ:𝒮→Δ​(𝒪) is the emission distribution, r:𝒮×𝒜→ℝ is the reward function, μ∈Δ​(𝒮) is the initial state distribution, and γ∈\[0,1) is the discount factor. Since the agent does not have direct access to st, it instead receives a partial observation ot∈𝒪 emitted by the environment, where ot∼𝒪(⋅∣st) is the emission distribution. The agent interacts with the environment over T turns. At each turn t∈{1,…,T}, the environment is in state st∈𝒮, from which the agent receives observation ot∈𝒪 and selects an action at∼π(⋅∣ht) based on the interaction history:

ht\=(o1,a1,…,ot−1,at−1,ot).

The environment then transitions to the next state st+1∼P(⋅∣st,at) and emits a reward rt\=r​(st,at). A complete trajectory τ\=(o1,a1,r1,…,oT,aT,rT) has discounted return R​(τ)\=∑t\=1Tγt−1​rt, and induces the trajectory distribution:

pπ​(τ)\=μ​(s1)​∏t\=1Tπ​(at∣ht)​P​(st+1∣st,at).

The value of policy π is thus J​(π)\=𝔼τ∼pπ​\[R​(τ)\].

### 3.2 Off-Policy Evaluation for LLM Agents

We assume access to an offline dataset:

𝒟\={τ(i)}i\=1N,τ(i)∼pπb​(τ),

collected by a behavior policy πb, which may correspond to a mixture of different LLM configurations and is treated as unknown. Given 𝒟 and a target evaluation policy πe, a language model distinct from πb that maps observation history ht to a distribution over text actions, the goal of off-policy evaluation (OPE) is to estimate:

J​(πe)\=𝔼τ∼pπe​\[R​(τ)\],

without executing πe in the real environment. The core challenge is distribution shift: since pπb​(τ)≠pπe​(τ), trajectories in 𝒟 are not representative samples from pπe, and direct Monte Carlo estimation on 𝒟 is severely biased. We address this challenge by learning a diffusion world model that simulates rollouts under πe directly, bypassing the need for explicit density ratio estimation.

### 3.3 Denoising Diffusion Models

Denoising Diffusion Probabilistic Models (DDPMs) (Ho et al., [2020](#bib.bib1 "Denoising diffusion probabilistic models")) learn to approximate a target distribution q​(x) via two coupled Markov chains. The forward process gradually corrupts a clean sample x0∼q​(x) over K steps:

xk\=α¯k​x0+1−α¯k​ϵ,ϵ∼𝒩​(0,I),

where {α¯k}k\=1K is a decreasing noise schedule with α¯K≈0. The reverse process learns to recover x0 from xK by iteratively applying pθ​(xk−1∣xk)\=𝒩​(xk−1;μθ​(xk,k),σk2​I), where μθ is parameterized via a noise prediction network ϵθ trained by:

ℒ​(θ)\=𝔼k,x0,ϵ∼𝒩​(0,I)​\[‖ϵ−ϵθ​(xk,k)‖2\].

#### Guided diffusion.

A trained diffusion model can be steered to sample from a conditional distribution p​(x∣y) without retraining, by incorporating a guidance signal y into each denoising step (Dhariwal and Nichol, [2021](#bib.bib2 "Diffusion models beat gans on image synthesis")). Since the backward transition is well approximated by a Gaussian, p​(xk∣xk+1)≈𝒩​(μk,Σk), Bayes’ rule gives:

p​(xk∣xk+1,y)∝p​(xk∣xk+1)​p​(y∣xk).

Applying a first-order Taylor expansion of log⁡p​(y∣xk) around μk yields:

p​(xk∣xk+1,y)≈𝒩​(xk;μk+Σk​gk,Σk),gk\=∇xlog⁡p​(y∣x)|x\=μk.

Sampling from the guided distribution therefore reduces to shifting the denoising mean by Σk​gk at each step, with no modification to the model itself. In our framework, y is instantiated as the target LLM agent πe, and gk becomes the log-likelihood gradient of πe with respect to the current latent state, steering world-model rollouts toward trajectories compatible with the agent under evaluation.

## 4 Autoregressive Diffusion World Models

We develop an autoregressive diffusion world model for off-policy evaluation of LLM agents. The core insight is that a globally policy-guided trajectory law can be exactly factored into a sequence of single-step conditionals, each fusing local action consistency with global policy guidance. We realize these conditionals as a guided diffusion process in a structured latent space, where the evaluation LLM agent directly steers the denoising at every step (Figure [2](#S4.F2 "Figure 2 ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")).

### 4.1 From Trajectory Guidance to Single-Step Conditionals

In our agentic setting, the behavior policy πb is unknown and generally intractable, as trajectories in 𝒟 may originate from different LLM configurations, prompting strategies, or agent versions. A natural starting point is importance sampling, which corrects for the mismatch between pπb and pπe via policy-likelihood ratios. Since the environment transition cancels between pπe and pπb, the target trajectory law satisfies:

pπe​(τ)\=pπb​(τ)⋅∏t\=1Tπe​(at∣ht)πb​(at∣ht),

(1)

so log⁡pπe​(τ)\=log⁡pπb​(τ)+∑tlog⁡πe​(at∣ht)−∑tlog⁡πb​(at∣ht). Since πb is unavailable, and dropping it acts as a form of behavior regularization (Jackson et al., [2024](#bib.bib4 "Policy-guided diffusion")) that anchors the distribution to the support of 𝒟, we learn a generative model pθ​(τ) of the logged trajectory distribution via maximum likelihood, and introduce α\>0 to control the correction strength, yielding the policy-guided target law:

qα​(τ∣πe)∝pθ​(τ)​exp⁡(α​∑t\=1Tlog⁡πe​(at∣ht)),α\>0.

(2)

When pθ\=pπb and α\=1, this recovers pπe exactly; the full derivation is in Appendix [C](#A3 "Appendix C Relation to Classical Importance Sampling ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents").

The target law ([2](#S4.E2 "Equation 2 ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) is defined over full trajectories, but cannot be instantiated directly for a black-box πe. Since πe is a black-box LLM that must observe ot before producing at, the trajectory-level tilt ∑tlog⁡πe​(at∣ht) cannot be evaluated without first generating all observations in causal order. This forces an autoregressive decomposition: the world model and πe must alternate step by step, generating observations and actions one at a time. The following proposition, proved in Appendix [B](#A2 "Appendix B Proof of Proposition 1 ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents"), characterizes the exact single-step conditional induced by ([2](#S4.E2 "Equation 2 ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")).

###### Proposition 1 (Autoregressive conditional law).

Let qα be the policy-guided trajectory law ([2](#S4.E2 "Equation 2 ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")). Then for every (ht,at):

qα​(ot+1∣ht,at;πe)∝pθ​(ot+1∣ht,at)​𝒞α​(ot+1;ht,at,πe),

(3)

where the _continuation factor_ is:

𝒞α​(ot+1;ht,at,πe):=𝔼τt+1:T∼pθ(⋅∣ht,at,ot+1)​\[exp⁡(α​∑u\=t+1Tlog⁡πe​(au∣hu))\].

(4)

Two candidates for ot+1 may be equally supported by pθ​(ot+1∣ht,at) yet lead to futures with very different compatibility with πe; 𝒞α is precisely the term that distinguishes them. A Bayesian decomposition of pθ​(ot+1∣ht,at) further separates the prior dynamics from the influence of at, giving the central result of our framework.

###### Theorem 1 (Fused single-step conditional).

Let qα be the policy-guided trajectory law ([2](#S4.E2 "Equation 2 ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")). For every (ht,at) and any ω\>0, η≥0:

q~ω,η​(ot+1∣ht,at;πe)∝Pθ​(ot+1∣ht)⏟prior⋅Pθ​(at∣ot+1,ht)ω⏟action posterior⋅𝒞α​(ot+1;ht,at,πe)η⏟policy continuation,

(5)

where ω\=η\=1 recovers the exact conditional induced by qα.

###### Remark 1 (Limiting cases of ω and η).

The parameters ω and η interpolate between degenerate regimes that illuminate the role of each guidance term. Setting η\=0 removes the policy continuation factor entirely, reducing q~ω,0 to an action-posterior-weighted prior that simulates the behavior distribution with no correction toward πe. Setting ω\=0 suppresses the action posterior, causing the model to generate observations from the unconditional prior Pθ​(ot+1∣ht) irrespective of at and breaking action-consistency. The canonical choice ω\=η\=1 recovers the exact conditional induced by qα. In practice, ω is controlled by the CFG scale λ via ω\=1+λ, and η is annealed over the reverse process; both are treated as hyperparameters rather than tuned per environment.

The three factors play distinct roles. The prior Pθ​(ot+1∣ht) anchors generated observations to the support of the offline data. The action posterior Pθ​(at∣ot+1,ht)ω injects current-step action information without a separate classifier, recovered directly from classifier-free guidance as shown in Section [4.2](#S4.SS2 "4.2 Guided Diffusion World Model ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents"). The continuation factor 𝒞α re-weights candidate observations by their long-horizon compatibility with πe, ensuring the world model generates observations that open up futures where πe can act effectively. Crucially, since ([5](#S4.E5 "Equation 5 ‣ Theorem 1 (Fused single-step conditional). ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) is a single-step conditional, πe never needs to produce actions before observations are generated, fully resolving the circular dependency that prevents existing diffusion OPE methods from operating autoregressively. The proof is given in Appendix LABEL:app:fusion. Each factor corresponds to a distinct gradient term in the score function, which we now realize as a guided diffusion process in latent space.

Since ([5](#S4.E5 "Equation 5 ‣ Theorem 1 (Fused single-step conditional). ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) is a product of three factors, its log decomposes into a sum, and differentiating with respect to ot+1 yields an additive score:

∇ot+1log⁡q~ω,η\=∇log⁡Pθ​(ot+1∣ht)⏟prior score+ω​∇log⁡Pθ​(at∣ot+1,ht)⏟action-posterior score+η​∇log⁡𝒞α​(ot+1;ht,at,πe)⏟continuation score.

(6)

This additive structure is precisely what makes guided diffusion applicable: each term can be approximated and injected independently into the reverse process without retraining the world model. The prior score is given directly by the score network; the action-posterior score is recovered from the difference between action-conditioned and unconditional outputs via classifier-free guidance; and the continuation score requires differentiating through πe’s log-likelihoods along future rollouts. We realize all three in a structured latent space, as detailed next.

### 4.2 Guided Diffusion World Model

The product structure of Theorem [1](#Thmtheorem1 "Theorem 1 (Fused single-step conditional). ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents") maps directly onto guided diffusion: each factor contributes an additive term to the score, and sampling reduces to running a guided reverse process with πe steering the denoising at every step. To instantiate this in practice, we first specify the latent space in which the diffusion operates and how πe reads world-model states.

#### Latent state and policy interface.

Observations in LLM agent tasks are raw text, which cannot be directly denoised. We therefore operate in a structured latent space. An encoder E maps each observation to a compact representation zt\=E​(ot)∈ℝd, and a projector Gψ produces soft tokens o~t\=Gψ​(zt)∈ℝK×dLLM that πe can read in its own embedding space. The world model operates on the latent history ht\=(z1,a1,…,zt), while πe conditions on htπ\=(o~1,a1,…,o~t). A pretrained semantic encoder is insufficient for this role: two observations that differ only in a negation may be nearby in semantic space yet lead to entirely different dynamics, corrupting all three guidance terms in ([9](#S4.E9 "Equation 9 ‣ Guided score. ‣ 4.2 Guided Diffusion World Model ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) simultaneously. We therefore train E end-to-end jointly with all other components on 𝒟, so that the representation is shaped by the dynamics of the environment rather than surface semantics. The projector Gψ is trained with a contrastive InfoNCE objective (Oord et al., [2018](#bib.bib51 "Representation learning with contrastive predictive coding")):

ℒψ\=−12​B​∑i\=1B\[log⁡esi​i/τ∑jesi​j/τ+log⁡esi​i/τ∑jesj​i/τ\]+βB​∑i\=1B‖G¯ψ​(zi)−e¯i‖22,

(7)

where si​j\=cossim​(G¯ψ​(zi),e¯j), G¯ψ​(zi) is the token-mean of the projected soft tokens, e¯i is the token-mean of the ground-truth πe input embeddings, τ is a temperature, and β\=0.1 balances discriminability against distributional alignment with πe’s embedding space. Two auxiliary losses shape the representation: an inverse dynamics loss ℒIDM encouraging action-aware latents, and a behavior cloning loss ℒBC providing a stable retrieval target for Gψ; full definitions are in [Section˜E.7](#A5.SS7 "E.7 World-model training objective ‣ Appendix E Experimental Setup Details ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents").

#### Guided score.

Following classifier-free guidance (Ho and Salimans, [2022](#bib.bib3 "Classifier-free diffusion guidance")), we parameterize the world model as a score network sθ​(zt+1k,k;htz,at). To realize the prior and action-posterior factors of Theorem [1](#Thmtheorem1 "Theorem 1 (Fused single-step conditional). ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents") without a classifier, we train sθ under two regimes by randomly masking at with probability p during training:

ℒDSM(θ)\=𝔼\[∥sθ(zt+1k,k;htz,a~t)−∇zklogpk(zt+1k∣zt+1)∥2\],a~t\={∅w.p. ​p,atotherwise.

(8)

By Bayes’ rule, the difference between the action-conditioned and unconditional outputs recovers the action-posterior score ∇log⁡pϕ​(at∣zt+1k,htz), so classifier-free guidance with scale λ directly realizes the first two terms of ([5](#S4.E5 "Equation 5 ‣ Theorem 1 (Fused single-step conditional). ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")), with ω\=1+λ.

For the continuation term, the score ∇zt+1log⁡𝒞α requires an expectation over all future trajectories and is intractable. At each diffusion step k, the score network provides a denoised estimate z^t+1; we roll the world model forward from this estimate and differentiate through πe’s log-likelihoods along the resulting trajectory. The complete guided score is:

s​(zt+1k,k;htz,at,πe)

\=∇zt+1klog⁡pθ​(zt+1k∣htz)+ω​∇zt+1klog⁡pϕ​(at∣zt+1k,htz)

(9)

#### Value estimation.

Given the guided transition model, we estimate J​(πe) by Monte Carlo rollout. At each step, πe samples an action at∼πe(⋅∣htπ), the world model generates the next latent zt+1 via the reverse diffusion process with score ([9](#S4.E9 "Equation 9 ‣ Guided score. ‣ 4.2 Guided Diffusion World Model ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")), and the reward head predicts r^t\=rρ​(zt,at,zt+1,htz). Episodes terminate when the termination head d^t\=σ​(dρ​(zt+1,htz)) exceeds a threshold. Averaging over M rollouts gives:

J^​(πe)\=1M​∑m\=1M∑t\=1Tmr^t(m),

(10)

where Tm is the termination step of the m\-th rollout. No environment interaction is required.

## 5 Experiments

We evaluate Adwm around three questions:

1.  1.
    
    _Can a world model rank unseen policies from behavior data alone?_
    
2.  2.
    
    _How does it compare to classical OPE under realistic LLM importance ratios?_
    
3.  3.
    
    _Which components of the diffusion guidance matter?_
    

#### Experiment setup.

We evaluate Adwm on four LLM-agent benchmarks chosen to span the reward and post-training regimes of modern LLM agents. On the reward axis, the suite covers dense per-step reward (HotpotQA F1 (Yang et al., [2018](#bib.bib47 "HotpotQA: a dataset for diverse, explainable multi-hop question answering"))), shaped partial reward (ScienceWorld (Wang et al., [2022](#bib.bib48 "Scienceworld: is your agent smarter than a 5th grader?"))), continuous partial reward (WebShop (Yao et al., [2022](#bib.bib5 "React: synergizing reasoning and acting in language models"))), and sparse binary success (ALFWorld (Shridhar et al., [2020](#bib.bib49 "Alfworld: aligning text and embodied environments for interactive learning"))). On the policy axis, the cells cover task-specific RLHF-style post-training (DPO and PRM on HotpotQA (Xiong et al., [2025](#bib.bib45 "Rag-gym: optimizing reasoning and search agents with process supervision")); ETO on ScienceWorld (Song et al., [2024](#bib.bib44 "Trial and error: exploration-based trajectory optimization of llm agents"))) and generic-to-specialized iterative fine-tuning (LEAP on ALFWorld and WebShop (Choudhury and Sodhi, [2024](#bib.bib46 "Better than your teacher: llm agents that learn from privileged ai feedback"))). In every cell, the behavior policy πb used to collect world-model training trajectories is strictly distinct from the evaluation policy πe being scored. Concretely, πb is either the matching task-specific SFT base of the same agent (HotpotQA, ScienceWorld) or, when no such base has been publicly released, the generic instruction-tuned Llama-3.1-8B-Instruct (Grattafiori et al., [2024](#bib.bib50 "The llama 3 herd of models")) (ALFWorld, WebShop). The world model is therefore trained without ever observing πe. On ALFWorld we additionally evaluate the third LEAP iteration to test cross-iteration ranking; on HotpotQA we pool DPO and PRM checkpoints into a 10-cell cross-policy split that probes policy-level discrimination beyond the within-policy ε\-axis.

To assess sensitivity across a graded range of policy quality, we construct a one-parameter family of evaluation policies via ε\-greedy mixing: at each step, πe acts with probability 1−ε and a uniformly random admissible action is taken with probability ε, for ε∈{0,0.25,0.5,0.75,1.0}. The real-environment ground-truth curve is obtained by executing πeε in the actual benchmark; Adwm produces J^ entirely from world-model rollouts with no environment access at evaluation time. We run five random seeds per cell and report the Spearman rank correlation between the seed-averaged J^ curve and the ground-truth curve. Full experimental details are in [Appendix˜E](#A5 "Appendix E Experimental Setup Details ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents").

#### Baselines.

We compare Adwm against five classical OPE estimators: direct method (DM) (Voloshin et al., [2019](#bib.bib33 "Empirical study of off-policy policy evaluation for reinforcement learning")), importance sampling (IS) (Precup et al., [2000](#bib.bib8 "Eligibility traces for off-policy policy evaluation")), weighted importance sampling (WIS) (Precup et al., [2000](#bib.bib8 "Eligibility traces for off-policy policy evaluation")), fitted-Q evaluation (FQE) (Le et al., [2019](#bib.bib22 "Batch policy learning under constraints")), and doubly robust estimation (DR) (Jiang and Li, [2016](#bib.bib11 "Doubly robust off-policy value evaluation for reinforcement learning")), all implemented following the COBS reference protocol (Voloshin et al., [2019](#bib.bib33 "Empirical study of off-policy policy evaluation for reinforcement learning")). For estimators that require πb (IS, WIS, DR), we supply exact per-token log-probabilities of the behavior LLM rather than the uniform action prior commonly used in prior work, providing these baselines with privileged information that is unavailable in realistic deployments (closed-source APIs, deleted checkpoints, hidden tokenizers). We include them as diagnostic references rather than competitive baselines, since the unbounded LLM action space and limited distributional overlap in long-horizon rollouts violate the absolute-continuity assumption underlying IS-based estimators. Implementation details and the per-ε FQE correction are in [Appendix˜F](#A6 "Appendix F Baseline Implementation Details ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents").

#### Training convergence.

[Figure˜3](#S5.F3 "In Training convergence. ‣ 5.1 Results ‣ 5 Experiments ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents") shows that all loss components converge stably across all four benchmarks within 50 epochs. Beyond ℒDSM ([8](#S4.E8 "Equation 8 ‣ Guided score. ‣ 4.2 Guided Diffusion World Model ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")), the total objective ℒWM includes two auxiliary terms: an inverse dynamics loss ℒIDM that predicts the bridging action between consecutive latents, encouraging action-aware representations, and a behavior cloning loss ℒBC that provides a stable retrieval target for the ψ adapter. ℒDSM converges fastest; ℒIDM and ℒBC converge more gradually, consistent with the greater difficulty of predicting actions and cloning behavior on long-horizon text observations. #### Ablations.

[Figure˜4](#S5.F4 "In Ablations. ‣ 5.1 Results ‣ 5 Experiments ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents") isolates the contribution of Adwm’s three core components across three environments with distinct reward structures. Each component dominates in a different reward regime, and no single component suffices across the full suite. Local CFG is most critical in sparse-reward settings: removing it collapses WebShop ρ from +0.90 to +0.10 and drops ScienceWorld from +1.00 to +0.60, while leaving HotpotQA intact (+0.90→+0.70). Continuation guidance is the primary mechanism for keeping trajectories on a successful path when per-step reward is absent, with removal degrading WebShop to +0.30 and ScienceWorld to +0.90. The ψ adapter dominates on linguistically rich environments: its removal drops HotpotQA to +0.40 and degrades both ScienceWorld and WebShop to +0.60. Adwm’s robustness therefore requires the combination of all three components. Additional ablations on behavior-data diversity, latent dimension, and ψ adapter loss design are in [Appendix˜H](#A8 "Appendix H Additional Ablations ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents").

## 6 Conclusion

We presented Adwm, an offline evaluation framework that estimates LLM agent policy value purely from pre-collected trajectories. The key contribution is an exact decomposition of a policy-guided trajectory law into autoregressive single-step conditionals, each realized as a guided diffusion process in which the evaluation policy steers every denoising step without retraining or explicit importance weights. Empirically, Adwm is the only estimator with consistently positive rank correlations across all six (πb,πe) configurations, with ablations confirming that robustness requires the combination of all three guidance components.

## Appendix A Limitations

Adwm has two practical scope conditions. First, like all OPE estimators, it is bottlenecked by ground-truth quality: on WebShop the binomial standard error on GT success rate (≈7%) is comparable to the gap between adjacent LEAP iterates (3–19%), so finer cross-iterate distinctions require substantially more GT episodes. Second, the ψ adapter is tied to the evaluation LLM’s embedding space, making cross-family evaluation (e.g., a Llama-trained world model scoring a Qwen-family πe) require a one-time adapter retrain. Neither condition reflects a fundamental limitation of the underlying formulation.

## Appendix C Relation to Classical Importance Sampling

We show that the policy-guided target law ([2](#S4.E2 "Equation 2 ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) is the natural generalization of importance sampling to the setting where the behavior policy πb is unknown, and that it recovers the evaluation trajectory law exactly when pθ\=pπb and α\=1.

#### Classical importance sampling identity.

For any two policies πb and πe, the induced trajectory laws satisfy:

pπe​(τ)

\=μ​(o1)​∏t\=1Tπe​(at∣ht)​P​(ot+1∣ht,at)

\=μ​(o1)​∏t\=1Tπb​(at∣ht)​P​(ot+1∣ht,at)⋅∏t\=1Tπe​(at∣ht)πb​(at∣ht)

\=pπb​(τ)⋅∏t\=1Tπe​(at∣ht)πb​(at∣ht),

(17)

where the environment transition P​(ot+1∣ht,at) cancels between numerator and denominator. Taking logarithms,

log⁡pπe​(τ)\=log⁡pπb​(τ)+∑t\=1Tlog⁡πe​(at∣ht)−∑t\=1Tlog⁡πb​(at∣ht).

(18)

Equation ([18](#A3.E18 "Equation 18 ‣ Classical importance sampling identity. ‣ Appendix C Relation to Classical Importance Sampling ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) is the starting point shared by importance sampling, policy-guided diffusion \[Jackson et al., [2024](#bib.bib4 "Policy-guided diffusion")\], and STITCH-OPE: the target trajectory law differs from the behavior trajectory law by a trajectory-level policy-likelihood correction.

#### Why πb is unavailable.

In our agentic setting, the offline dataset 𝒟 may aggregate trajectories from heterogeneous sources—different prompting strategies, decoding configurations, or agent versions. The behavior policy πb is therefore a mixture whose density πb​(at∣ht) is generally unavailable in closed form, preventing direct instantiation of ([17](#A3.E17 "Equation 17 ‣ Classical importance sampling identity. ‣ Appendix C Relation to Classical Importance Sampling ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")).

#### From IS to the policy-guided target law.

We replace the unknown pπb​(τ) with a learned generative model pθ​(τ) trained on 𝒟 by maximum likelihood. Dropping the negative behavior term −∑tlog⁡πb​(at∣ht) from ([18](#A3.E18 "Equation 18 ‣ Classical importance sampling identity. ‣ Appendix C Relation to Classical Importance Sampling ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) acts as a form of behavior regularization \[Jackson et al., [2024](#bib.bib4 "Policy-guided diffusion")\]: it anchors the target distribution to the support of the offline data rather than allowing unbounded correction away from it. Exponentiating and introducing a strength parameter α\>0 yields:

qα​(τ∣πe)∝pθ​(τ)​exp⁡(α​∑t\=1Tlog⁡πe​(at∣ht)),

(19)

which is ([2](#S4.E2 "Equation 2 ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) in the main text.

#### Recovery of pπe at pθ\=pπb, α\=1.

Suppose the learned prior is exact, i.e. pθ\=pπb. Then ([19](#A3.E19 "Equation 19 ‣ From IS to the policy-guided target law. ‣ Appendix C Relation to Classical Importance Sampling ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) with α\=1 gives:

q1​(τ∣πe)

∝pπb​(τ)​exp⁡(∑t\=1Tlog⁡πe​(at∣ht))

\=pπb​(τ)​∏t\=1Tπe​(at∣ht).

(20)

The normalizing constant of ([20](#A3.E20 "Equation 20 ‣ Recovery of 𝑝_𝜋_𝑒 at 𝑝_𝜃=𝑝_𝜋_𝑏, 𝛼=1. ‣ Appendix C Relation to Classical Importance Sampling ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) is:

Z

\=∫pπb​(τ)​∏t\=1Tπe​(at∣ht)​d​τ

\=∫μ​(o1)​∏t\=1Tπb​(at∣ht)​P​(ot+1∣ht,at)⋅∏t\=1Tπe​(at∣ht)​d​τ.

(21)

Using ([17](#A3.E17 "Equation 17 ‣ Classical importance sampling identity. ‣ Appendix C Relation to Classical Importance Sampling ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) to substitute pπb​(τ)​∏tπe​(at∣ht)/πb​(at∣ht)\=pπe​(τ):

Z

\=∫pπb​(τ)​∏t\=1Tπe​(at∣ht)πb​(at∣ht)⋅∏t\=1Tπb​(at∣ht)​d​τ

\=∫pπe​(τ)​𝑑τ\= 1.

(22)

Therefore ([20](#A3.E20 "Equation 20 ‣ Recovery of 𝑝_𝜋_𝑒 at 𝑝_𝜃=𝑝_𝜋_𝑏, 𝛼=1. ‣ Appendix C Relation to Classical Importance Sampling ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) is already normalized, and:

q1​(τ∣πe)|pθ\=pπb\=pπb​(τ)​∏t\=1Tπe​(at∣ht)\=pπe​(τ),

(23)

where the last equality follows from ([17](#A3.E17 "Equation 17 ‣ Classical importance sampling identity. ‣ Appendix C Relation to Classical Importance Sampling ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")). This confirms that qα with α\=1 and pθ\=pπb recovers the evaluation trajectory law exactly.

#### Role of α.

For α≠1 or pθ≠pπb, qα​(τ∣πe) interpolates between the offline prior (α→0, recovering pθ) and a distribution that increasingly concentrates on trajectories preferred by πe (α→∞). In practice, α is a hyperparameter that balances the fidelity of the simulated environment against the alignment of generated rollouts with the evaluation policy.

## Appendix D Derivation of the Guided Score Function

We derive ([9](#S4.E9 "Equation 9 ‣ Guided score. ‣ 4.2 Guided Diffusion World Model ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) in three steps: (i) decompose the score of the fused conditional into three additive terms; (ii) show that the first two terms are realized by classifier-free guidance; (iii) derive the tractable approximation of the continuation score.

### D.1 Score decomposition of the fused conditional

By Theorem [1](#Thmtheorem1 "Theorem 1 (Fused single-step conditional). ‣ 4.1 From Trajectory Guidance to Single-Step Conditionals ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents"), the target single-step conditional is:

q~ω,η​(zt+1∣h¯t,at;πe)∝pθ​(zt+1∣h¯t)1⋅pϕ​(at∣zt+1,h¯t)ω⋅𝒞α​(zt+1;h¯t,at,πe)η.

(24)

Taking logarithms:

log⁡q~ω,η​(zt+1∣h¯t,at;πe)\=log⁡pθ​(zt+1∣h¯t)+ω​log⁡pϕ​(at∣zt+1,h¯t)+η​log⁡𝒞α​(zt+1;h¯t,at,πe)+const,

(25)

where the constant absorbs the normalizer, which does not depend on zt+1. Differentiating with respect to zt+1 gives the exact score:

∇zt+1log⁡q~ω,η\=∇zt+1log⁡pθ​(zt+1∣h¯t)⏟prior score+ω​∇zt+1log⁡pϕ​(at∣zt+1,h¯t)⏟action-posterior score+η​∇zt+1log⁡𝒞α​(zt+1;h¯t,at,πe)⏟continuation score.

(26)

Each term has a distinct role: the prior score keeps generated latents on the manifold of the offline data; the action-posterior score steers generation toward latents from which at is recoverable; the continuation score steers generation toward latents that open up futures compatible with πe.

### D.2 Realization of the prior and action-posterior scores via classifier-free guidance

We show that the first two terms of ([26](#A4.E26 "Equation 26 ‣ D.1 Score decomposition of the fused conditional ‣ Appendix D Derivation of the Guided Score Function ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) are jointly realized by classifier-free guidance \[Ho and Salimans, [2022](#bib.bib3 "Classifier-free diffusion guidance")\] without a separate classifier.

Consider the joint density pθ​(zt+1,at∣h¯t). By the chain rule it factors in two ways:

pθ​(zt+1,at∣h¯t)

\=pθ​(zt+1∣h¯t,at)​pθ​(at∣h¯t),

(27)

pθ​(zt+1,at∣h¯t)

\=pϕ​(at∣zt+1,h¯t)​pθ​(zt+1∣h¯t).

(28)

Equating ([27](#A4.E27 "Equation 27 ‣ D.2 Realization of the prior and action-posterior scores via classifier-free guidance ‣ Appendix D Derivation of the Guided Score Function ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) and ([28](#A4.E28 "Equation 28 ‣ D.2 Realization of the prior and action-posterior scores via classifier-free guidance ‣ Appendix D Derivation of the Guided Score Function ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) and dividing both sides by pθ​(at∣h¯t)\>0:

pθ​(zt+1∣h¯t,at)\=pϕ​(at∣zt+1,h¯t)​pθ​(zt+1∣h¯t)pθ​(at∣h¯t).

(29)

Taking logarithms and differentiating with respect to zt+1, noting that pθ​(at∣h¯t) is constant in zt+1:

∇zt+1log⁡pθ​(zt+1∣h¯t,at)\=∇zt+1log⁡pθ​(zt+1∣h¯t)+∇zt+1log⁡pϕ​(at∣zt+1,h¯t).

(30)

Rearranging:

∇zt+1log⁡pϕ​(at∣zt+1,h¯t)\=∇zt+1log⁡pθ​(zt+1∣h¯t,at)−∇zt+1log⁡pθ​(zt+1∣h¯t).

(31)

Substituting ([31](#A4.E31 "Equation 31 ‣ D.2 Realization of the prior and action-posterior scores via classifier-free guidance ‣ Appendix D Derivation of the Guided Score Function ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) into the first two terms of ([26](#A4.E26 "Equation 26 ‣ D.1 Score decomposition of the fused conditional ‣ Appendix D Derivation of the Guided Score Function ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")):

∇zt+1log⁡pθ​(zt+1∣h¯t)+ω​∇zt+1log⁡pϕ​(at∣zt+1,h¯t)

\=∇zt+1log⁡pθ​(zt+1∣h¯t)+ω​(∇zt+1log⁡pθ​(zt+1∣h¯t,at)−∇zt+1log⁡pθ​(zt+1∣h¯t))

\=(1−ω)​∇zt+1log⁡pθ​(zt+1∣h¯t)+ω​∇zt+1log⁡pθ​(zt+1∣h¯t,at).

(32)

Writing ω\=1+λ, ([32](#A4.E32 "Equation 32 ‣ D.2 Realization of the prior and action-posterior scores via classifier-free guidance ‣ Appendix D Derivation of the Guided Score Function ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) becomes:

(1+λ)​∇zt+1log⁡pθ​(zt+1∣h¯t,at)−λ​∇zt+1log⁡pθ​(zt+1∣h¯t),

(33)

which is exactly the classifier-free guidance combination with scale λ. In practice, both scores are approximated by the action-conditioned and unconditional branches of the shared score network sθ, trained jointly via ([8](#S4.E8 "Equation 8 ‣ Guided score. ‣ 4.2 Guided Diffusion World Model ‣ 4 Autoregressive Diffusion World Models ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) with action dropout.

### D.3 Tractable approximation of the continuation score

The continuation factor is defined as:

𝒞α​(zt+1;h¯t,at,πe):=𝔼τt+1:T∼pθ(⋅∣h¯t,at,zt+1)​\[exp⁡(α​∑u\=t+1Tlog⁡πe​(au∣h~u))\].

(34)

Computing ∇zt+1log⁡𝒞α exactly requires differentiating through an expectation over all future trajectories, which is intractable. We derive a tractable approximation via the denoised estimate.

At diffusion step k, the score network provides the posterior mean estimate:

z^t+1\=𝔼​\[zt+1∣zt+1k\]\=zt+1k−1−α¯k​sθ​(zt+1k,k;h¯t,at)α¯k.

(35)

We substitute z^t+1 for zt+1 in ([34](#A4.E34 "Equation 34 ‣ D.3 Tractable approximation of the continuation score ‣ Appendix D Derivation of the Guided Score Function ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) and replace the expectation over future trajectories with a single greedy rollout τ^t+1:T\=ℛθ​(z^t+1;h¯t,at):

log⁡𝒞α​(zt+1;h¯t,at,πe)≈α​∑u\=t+1Tlog⁡πe​(a^u∣h~u).

(36)

Differentiating ([36](#A4.E36 "Equation 36 ‣ D.3 Tractable approximation of the continuation score ‣ Appendix D Derivation of the Guided Score Function ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) with respect to z^t+1 and applying the chain rule through ([35](#A4.E35 "Equation 35 ‣ D.3 Tractable approximation of the continuation score ‣ Appendix D Derivation of the Guided Score Function ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")) gives the tractable continuation gradient:

η​∇zt+1log⁡𝒞α≈ηk​α​∇z^t+1​∑u\=t+1Tlog⁡πe​(a^u∣h~u),

(37)

where we write ηk to allow the continuation weight to vary with noise level k, suppressing the signal when z^t+1 is unreliable at high noise and amplifying it late in the reverse process.

## Appendix E Experimental Setup Details

### E.1 Software stack and reproducibility

All training and OPE rollout code is implemented in PyTorch 2.1 (CUDA 12.1) under Python 3.10. LLM-side πb/πe inference is served by vLLM 0.5.4 with bf16 weights and a tensor-parallel size of 1 (one GPU per LLM job). Baselines (FQE, DR, IS, WIS, DM) are implemented to match the COBS reference protocol of Voloshin et al. \[[2019](#bib.bib33 "Empirical study of off-policy policy evaluation for reinforcement learning")\] verbatim except for the per-ε FQE correction described in [Section˜F.4](#A6.SS4 "F.4 FQE Q-network and per-𝜀 correction ‣ Appendix F Baseline Implementation Details ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents"). The world model itself is trained _once per environment_ with seed s0; the five per-cell seeds vary inference-time stochasticity only.

### E.5 ε\-greedy mixing protocol

Mixing is applied at the _probability_ level (not the log level) to ensure the resulting sampling distribution is a valid probability measure:

πeε​(a|s)\=(1−ε)​πe​(a|s)+ε​1|𝒜adm​(s)|​𝕀​\[a∈𝒜adm​(s)\],

(39)

where 𝒜adm​(s) is the set of admissible actions reported by the environment. We sample one Bernoulli flip per step: with probability 1−ε we forward the LLM-decoded action a∼πe(⋅|s); with probability ε we sample a∼Unif​(𝒜adm​(s)). Both Adwm imagined rollouts and the GT collection use this exact protocol with the same seed.

### E.6 World-model architecture

The world model is a 7.4M-parameter latent diffusion model with the modular structure shown in [Table˜4](#A5.T4 "In E.6 World-model architecture ‣ Appendix E Experimental Setup Details ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents"). The largest component is the text-based observation encoder (5.05 M params), which tokenises raw obs\_textt via a stable hash-bucket tokeniser (vocabulary size 32,768, max length 128 tokens) and embeds it into the latent dimension dz\=64 through a 4-layer TransformerEncoder of width 128 with 4 attention heads. The history encoder is a causal 4-layer Transformer with width 128 and a generated subsequent-token mask, conditioning on (zt,at) tuples. The score network is a 3-layer FiLM-MLP of width 256 with sinusoidal time embedding (dimension 64) and zero-initialised output projection; the IDM and BC heads are 2-layer MLPs over (z,h). All non-linearities are SiLU except the IDM / policy / projector heads, which use ReLU and GELU respectively.

### E.7 World-model training objective

The world model is trained with the multi-task objective

ℒWM\=ℒDSM+λidm​ℒIDM+λbc​ℒBC+λdiv​ℒdiv+λr​ℒr+λd​ℒd,

(40)

with weights (λidm,λbc,λdiv,λr,λd)\=(0.1, 0.1, 0.05, 10, 1). The component losses are defined as follows.

#### (a) Denoising score matching (ℒDSM).

For a sampled diffusion step k∼Unif​(1,T), latent z∼fθ​(obst+1), history h¯t, and action at, we form the noised latent zk\=α¯k​z+1−α¯k​ξ with ξ∼𝒩​(0,I) and a cosine α¯k schedule. We train the conditional score network ϵθ​(zk,k,h¯t,at) together with its unconditional counterpart ϵθ​(zk,k,h¯t,∅) under classifier-free dropout puncond\=0.1:

ℒDSM\=𝔼k,ξ​\[ω​(k)​(‖ϵθ​(zk,k,h¯t,at)−ξ‖22+‖ϵθ​(zk,k,h¯t,∅)−ξ‖22)\],ω​(k)\= 1−α¯k.

(41)

The ω​(k)\=1−α¯k weighting keeps the per-step loss numerically comparable across k.

#### (b) Inverse-dynamics regulariser (ℒIDM).

A discriminative auxiliary that predicts the bridging action between two consecutive latents: ℒIDM\=CE​(IDM​(zt+1,h¯t),at). This pushes z to be _action-aware_ and prevents the diffusion prior from collapsing onto policy-invariant features.

#### (c) Behavior-cloning soft-token loss (ℒBC).

ℒBC\=CE​(BC​(zt,h¯t),at). Trains the latent-to-policy interface against logged actions _independently_ of any specific evaluation policy, providing the ψ\-adapter retrieval target a stable signal.

#### (d) Diversity regulariser (ℒdiv).

A cross-trajectory contrastive penalty that prevents the latent space from collapsing across distinct trajectories. For batch {z1:L(i)}i\=1B, let z¯(i)\=normalize​(meant​zt(i)) be the per-trajectory pooled latent. Then

ℒdiv\=1B​(B−1)​∑i≠j|z¯(i)⊤​z¯(j)|.

(42)

Same-window adjacent latents are deliberately left free since the dynamics constraint requires them to stay close.

#### (e) Reward head (ℒr).

For environments with continuous reward (HotpotQA F1, ScienceWorld shaped reward, WebShop product match) we use MSE on sigmoid-clipped reward-to-go targets: ℒr\=𝔼t​\[(σ​(Rθ​(zt+1,h¯t))−rttgt)2\] with rttgt∈\[0,1\]. For sparse-reward ALFWorld we replace the MSE by a balanced binary cross-entropy with sigmoid output and per-batch positive-class weight min⁡(50,nneg/npos), which prevents the reward head from collapsing to the constant p≈r¯ predictor. The reward head is supervised on _terminal steps only_ when the environment provides a terminal-only success label, and on all steps when shaped per-step rewards are available.

#### (f) Termination head (ℒd).

A binary cross-entropy with class-balanced positive weight min⁡(50,nneg/npos): ℒd\=BCE​(Dθ​(zt+1,h¯t), 1​\[done at ​t+1\]). Used during imagined rollouts to terminate trajectories that the world model predicts have ended.

### E.9 ψ\-adapter training

The ψ adapter projects latent observations into a sequence of soft tokens that drop into the input embedding space of πe. The adapter Gψ:ℝdz→ℝnsoft×dllm is a 3-layer GELU MLP (dz→256→256→8⋅dllm) with nsoft\=8 soft tokens.

## Appendix F Baseline Implementation Details

All baselines follow the COBS reference protocol of Voloshin et al. \[[2019](#bib.bib33 "Empirical study of off-policy policy evaluation for reinforcement learning")\] unless stated otherwise. The full set of estimators is implemented in Adwm/ope/baselines.py and Adwm/ope/fqe\_dr.py.

### F.1 Behavior log-probability extraction

For estimators that require πb (IS, WIS, DR), we re-run the behavior LLM in inference mode and extract per-token log-probabilities at the action span of every recorded trajectory. The action span is identified by longest-common-token-prefix matching to handle BPE-merge boundaries between the prompt and action region (a common failure mode that produces off-by-one log-probabilities and silently inflates importance ratios). πe log-probabilities are obtained identically. We then mix at the probability level πeε​(a)\=(1−ε)​πe​(a)+ε/|𝒜adm| and take the log to form per-step log-ratios log⁡ρt\=log⁡πeε​(at|st)−log⁡πb​(at|st).

### F.3 Per-cell importance-ratio variance

Empirically observed cumulative log-ratio magnitudes at ε\=0.5 span tens of nats per trajectory (HotpotQA up to ±24, ScienceWorld ±18, ALFWorld ±35, WebShop ±28), corresponding to per-trajectory ratios that span ∼16–30 orders of magnitude. This is the regime in which IS and DR estimates fluctuate by 1020 or more even in double-precision arithmetic; our raw baseline output JSONs (released alongside the code) record DR values up to |JDR|≈6×1020 on HotpotQA-PRM, exemplifying this divergence. The corresponding rank-correlations become driven entirely by floating-point underflow ties rather than the underlying signal.

### F.4 FQE Q-network and per-ε correction

The FQE Q-network is a 2-hidden-layer MLP Qϕ:ℝdz+da→ℝ (dz+da→128→128→1, SiLU activations). A target network Qϕ′ is updated by Polyak averaging with τ\=0.99 (ϕ′←0.99​ϕ′+0.01​ϕ). optimization: Adam, lr 3×10−4, batch size 64, 20 epochs, discount γ\=0.99. Each training minibatch consists of (st,at,rt,st+1,donet) tuples drawn from 𝒟b, and the Bellman target is y\=rt+γ​(1−donet)​Vπeε​(st+1) with Vπeε​(s)\=(1−ε)​maxa⁡Qϕ′​(s,a)+ε​Q¯ϕ′​(s) where Q¯ϕ′​(s) is the action-mean across the admissible set. The maxa approximation is necessary because the LLM does not expose a full per-action probability vector; it is exact in the deterministic-greedy limit of πe that all our LLM-agent benchmarks effectively use (temperature 0 LLM decoding).

#### Per-ε correction.

Under the COBS protocol, mixing is applied at evaluation time as Vε​(s)\=(1−ε)​maxa⁡Qϕ′​(s,a)+ε​Q¯ϕ′​(s), producing a value linear in ε and trivially monotonically decreasing. This artefact gives FQE a spurious ρ\>0 on the ε\-axis even when its underlying state-value estimate is incorrect, and inflates the COBS-protocol headline number to a rank that has nothing to do with the policy being scored. We remove this artefact by training a separate Q-network for each ε level using πb trajectories filtered against the ε\-mixed behavior likelihood, then evaluating each on its own Vπeε​(s0) at the trajectory start. This is the _per-ε correction_ reported in the main text and is the only modification to FQE relative to COBS. Without it, FQE reports a near-uniform ρ\=+0.9 on every cell; with it, the true rank correlation collapses or inverts as reported in [Table˜2](#S5.T2 "In Main results. ‣ 5.1 Results ‣ 5 Experiments ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents").

Per-seed Spearman is computed by independently running each ε cell with seed si∈{0,1,2,3,4}, which propagates randomness through both imagined-rollout sampling and the ε\-greedy admissible-action choice. The avg-J^ aggregator used in [Table˜2](#S5.T2 "In Main results. ‣ 5.1 Results ‣ 5 Experiments ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents") mitigates this by averaging J^ across seeds before computing ρ.

## Appendix H Additional Ablations

### H.1 Five-seed extension of [Fig.˜4](#S5.F4 "In Ablations. ‣ 5.1 Results ‣ 5 Experiments ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents")

[Figure˜4](#S5.F4 "In Ablations. ‣ 5.1 Results ‣ 5 Experiments ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents") reports single-seed ρ values to keep the component story compact. For each variant we also re-ran the full 5-seed protocol on the same three environments and recovered the same qualitative ordering: removing local CFG drops WebShop ρ to +0.12±0.08 across seeds, removing continuation guidance drops it to +0.27±0.10, and removing the ψ adapter drops HotpotQA to +0.42±0.07. No 5-seed run flips the sign of the corresponding single-seed value, and the relative magnitude of each component’s contribution is preserved.

### H.2 Behavior-data and capacity ablations

Beyond the inference-time guidance and adapter terms ablated in [Fig.˜4](#S5.F4 "In Ablations. ‣ 5.1 Results ‣ 5 Experiments ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents"), two further training-side factors prove equally critical. We probe both on WebShop, the most data-sensitive of our cells:

*   •
    
    Behavior-pool diversity. Restricting the behavior LLM pool from two models (Qwen3-1.7B + Qwen3-4B) to a single Qwen3-1.7B collapses ρ from +0.90 to −1.00 — a complete inversion. The world model in this regime overfits to the single πb’s idiosyncratic action distribution, and the reward head learns features that anti-correlate with the GT ranking under any πe.
    
*   •
    
    Latent capacity. Halving the latent dimension from dz\=64 to dz\=32 produces the same complete inversion (ρ\=−1.00), suggesting the latent is at a representational floor below which contrastive features cannot be preserved.
    

These two factors define a representational pre-requisite for Adwm: adequate behavior-data diversity and adequate latent capacity must both be in place before any choice of guidance can recover positive ranking. We did not observe similar inversions on HotpotQA or ScienceWorld at the same parameter cuts, suggesting the WebShop result reflects the combination of long-horizon search and continuous partial reward rather than a generic property.

### H.3 ψ\-adapter loss design

[Table˜7](#A8.T7 "In H.3 𝜓-adapter loss design ‣ Appendix H Additional Ablations ‣ Autoregressive Diffusion World Models for Off-Policy Evaluation of LLM Agents") shows that the loss design we adopt (InfoNCE + 0.1× MSE) is essential. Pure MSE collapses to a near-constant adapter on every environment because per-token-mean target embeddings have low variance, providing the optimiser with no signal to differentiate latents; pure InfoNCE produces a discriminative adapter whose outputs drift out-of-distribution for the LLM and break its downstream input expectations. The combined loss outperforms either alone on all three environments.
